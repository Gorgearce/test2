Module One

Module 1 focuses on the basics of technology. This module introduces students to hardware and front-end web development. Students learn the major components necessary to build a computer. Students also learn CSS and HTML to create a website.

Module Two

Module 2 offers more in-depth topics in technology such as software and networking. Students learn how to troubleshoot using multiple operating systems and how to setup a LAN and WAN network.

Module Three

During Module 3, students are placed in different tracks based on their interest and learning style. The tracks Year Up offers include:

- IT
- Networking
- Quality Assurance
- Project Coordination
- Cyber Security

Module 3 also focuses heavily on internship preparation.
