Quality Assurance Track at Year Up

Topics Covered in Class

The Quality Assurance track at Year Up teaches students the basics of Quality Assurance and Software Testing. During the first few weeks, students learn about the SDLC (Software Development Life Cycle). The course teaches different testing methods such as black-box and white-box testing. The next few weeks cover bug testing and how to log a bug test. Tools like JIRA and other bug testing software is introduced in this course. Students also learn the coding language of Javascript.

The textbook used in the class is Software Testing 2nd Edition by Ron Patton.
